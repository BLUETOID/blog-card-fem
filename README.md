## Overview 

A responsive **blog preview card** built to match the Frontend Mentor design. This project helped me practice semantic HTML, responsive layout with Flexbox, and accessible interactive states (hover / focus).

## Features 

- Responsive layout for mobile and desktop
- Hover and keyboard focus states for interactive elements
- Semantic HTML structure and descriptive `alt` text for images
- Uses local fonts from the `/assets/fonts` folder

## Built with 

- HTML5 (semantic elements)
- CSS3 — Flexbox, CSS custom properties, mobile-first approach
- No JavaScript required


## What I learned 

- How to use Flexbox to align elements both vertically and horizontally.
- Why semantic HTML matters — it improves accessibility and keeps markup meaningful.
- The CSS box model and how padding/margin affect layout and spacing.
- The importance of hover and focus styles so interactive elements are usable by keyboard users.
